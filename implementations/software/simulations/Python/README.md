# Python Implementations — Ce Protocol (Software Layer)

## Purpose

The `python/` folder provides **reference implementations of Ce protocol logic in Python**.  
Python is chosen for **readability, experimentation, and rapid prototyping**, allowing developers to test ideas before deploying to hardware or other languages.

This folder is **non-normative** unless explicitly labeled otherwise.

---

## Scope

Contents typically include:

- Node simulations and examples
- API call demonstrations
- Event handling and subscription
- Recall propagation tests
- Coherence experiments
- Utility functions for network simulation or visualization

Python implementations **illustrate protocol behavior** but do not enforce production standards.
